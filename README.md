# Omini

Omini is a dynamic web application built to provide anime fans with easy access to popular anime series. Users can search for anime titles and explore details, trailers, and more. The website is responsive and designed to provide a smooth user experience across devices.

## Features
- Search for anime series
- Clean, responsive design
- User-friendly interface

## Technologies Used
- HTML
- CSS (Vanilla)
- JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lcc33/omini.git
   ```
2. Open the `index.html` file in your browser.
